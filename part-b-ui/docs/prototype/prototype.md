**→ [프로토타입 보기](https://claude.ai/public/artifacts/ef346d90-b364-482c-beff-0290ec0b8ed6)**

> 아래 프롬프트를 Claude에 입력하여 위 프로토타입을 생성하였습니다.

---

## 역할 / 맥락

B2B SaaS 서비스의 **조직 관리자(Admin)** 가 사용하는 설정 화면을 만들어주세요.
조직별로 문자 발송 업체(클라우드메시지 또는 센드톡)를 선택하고, 해당 업체의 API 연동 정보(Credential)를 입력·저장하는 화면입니다.

- 접근 권한: `role = ADMIN` (조직 관리자만 접근)
- 1개 조직 = 1개 업체 (단일 선택)
- 언어: 한국어
- 스타일: 깔끔한 B2B 대시보드 (Shadcn UI / Tailwind CSS 계열)
- 반응형: 데스크톱 우선 (min-width: 1280px)

## 업체별 Credential 입력 필드

### 클라우드메시지 선택 시

| API 필드명 | 레이블 | 타입 | 필수 | 플레이스홀더 |
|-----------|--------|------|------|-------------|
| `accessKey` | Access Key | 마스킹 입력 (눈 아이콘 토글) | ✅ | 클라우드메시지에서 발급받은 Access Key |
| `secretKey` | Secret Key | 마스킹 입력 (눈 아이콘 토글) | ✅ | 클라우드메시지에서 발급받은 Secret Key |
| `senderProfileId` | 발신 프로필 ID | 마스킹 입력 (눈 아이콘 토글) | ✅ | 알림톡 발신 채널 프로필 ID |
| `senderPhoneNumber` | 발신번호 | tel | ✅ | 010-0000-0000 |

### 센드톡 선택 시

| API 필드명 | 레이블 | 타입 | 필수 | 플레이스홀더 |
|-----------|--------|------|------|-------------|
| `apiKey` | API Key (Bearer Token) | 마스킹 입력 (눈 아이콘 토글) | ✅ | 센드톡에서 발급받은 API Key |
| `secretKey` | Secret Key | 마스킹 입력 (눈 아이콘 토글) | ✅ | Webhook 수신 결과 서명 검증용 |
| `senderPhoneNumber` | SMS 대체 발송용 발신번호 | tel | ✅ | 010-0000-0000 |

> 센드톡은 API Key 발급 시 카카오 채널이 계정에 고정 연결됩니다. 별도 채널 ID 입력란은 불필요합니다.

## 입력 규칙

- 마스킹 입력 필드: 기존 값이 있으면 `AKCL••••••••••••` (앞 4자리 + 마스킹) 표시. 클릭 시 초기화 후 재입력
- 모든 텍스트 필드: 붙여넣기 시 앞뒤 공백 자동 제거 (trim)
- 업체 전환 시: 입력 필드 전체 초기화

## 발신번호 유효성 검증

```
입력 중:  숫자 외 문자 차단, 하이픈 자동 삽입
포커스 이탈: 형식 검증 실행
에러 메시지: "올바른 발신번호 형식이 아닙니다. (예: 010-1234-5678)"
API 전송 시: 하이픈 제거 후 숫자만 전송
```

## 연결 테스트 결과 메시지

| providerErrorCode | 표시 메시지 |
|-------------------|------------|
| `AUTHENTICATION_FAILED` / `UNAUTHORIZED` | API Key가 올바르지 않습니다. |
| `INVALID_SIGNATURE` | Secret Key가 올바르지 않습니다. |
| `INVALID_SENDER` / `INVALID_PHONE_NUMBER` | 등록되지 않은 발신번호입니다. 업체에서 발신번호 등록 여부를 확인해주세요. |
| `QUOTA_EXCEEDED` / `RATE_LIMIT_EXCEEDED` | 요청 한도를 초과했습니다. 잠시 후 다시 시도해주세요. |
| `SERVICE_UNAVAILABLE` / `INTERNAL_ERROR` | 업체 서버에 일시적인 오류가 발생했습니다. 잠시 후 다시 시도해주세요. |
| 그 외 | 연결에 실패했습니다. 입력값을 다시 확인해주세요. |
