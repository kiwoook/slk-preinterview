# SLK 사전 인터뷰 과제

**현재 상태:** Part A 완료 | Part B 완료

## 과제 개요

B2B SaaS 서비스에서 조직(Organization)별로 서로 다른 메시지 전송 업체를 사용할 수 있도록 시스템을 변경하는 작업 계획을 설계합니다.

- 기존 업체: 클라우드메시지 (전체 조직 공통 사용 중)
- 신규 업체: 센드톡 (도입 예정)
- 지원 메시지 유형: SMS, MMS, 알림톡(카카오), 앱 푸시
